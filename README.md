# オクデラメディカル 貸しスタジオ - ウェブサイト

## プロジェクト概要
東京都北区王子のウエルネスオクデラビルズ3Fに入居する「貸しスタジオ」のQRコード連動型スマートフォン予約サイトです。

## 完成済み機能

### ページ構成
- **ヒーローセクション**: フラットイラスト調スタジオ画像（オーバーレイなし・くっきり表示）
- **施設特徴**: ロッカールーム・シャワー・音響・Wi-Fi・鏡張りの4カード
- **スタジオ写真**: 実写スタジオ内観とセミナールーム
- **プログラム案内 (アコーディオン3柱)**:
  - フィットネス・ウェルネス系（ヨガ、太極拳、ホットヨガ、ストレッチ）
  - ダンス・パフォーマンス系（バレエ、キッズ、社交ダンス、ヒップホップ、K-POP）
  - その他の活用（セミナー、会議、ワークショップ、パーソナルトレーニング）
- **料金プラン**: 時間利用・半日パック・1日パック・定期利用（要相談）
- **予約フォーム**: 3ステップ式（ご利用内容→日時→連絡先）。プログレスバー・入力サマリ表示付き
- **QRコード**: PC/タブレット表示時のみ自動生成（スマホでは省略し共有案内へ）
- **アクセス情報**: Googleマップ + 住所・電話・交通
- **FAQ**: アコーディオン形式6項目

### スマホ最適化
- **ハンバーガーメニュー**: 右上アイコンからドロワー表示。電話CTA・全セクションリンク
- **ヒーローCTA**: 縦並び・幅100%・最低高52pxでタップしやすく
- **QRセクション省略**: スマホ来訪者にはQR不要なため、代わりに共有案内を表示
- **ステップ式フォーム**: 1画面に詰めず3ステップ分割。各ステップでバリデーション
- **フローティングCTA**: 予約フォーム表示中は自動非表示（フォームの送信ボタンと重なるのを防止）
- **「もっと見る」展開**: スタジオ活用プログラムはPC=各カテゴリ3件 / スマホ=各カテゴリ1件を初期表示。「もっと見る（+N件）」で残りを展開。リサイズで自動再計算

## デプロイ

**Publish タブ**からウェブサイトを公開してください。公開後にQRコードが自動生成され、電光看板デザインに組み込むことができます。

## 次のステップ

1. **Publish タブ**からサイトを公開
2. 公開URLのQRコードをスクリーンショット
3. 電光看板の承認済みデザインにQRコードを追加
4. A1サイズ(83cm×63cm)で印刷用データを最終出力
